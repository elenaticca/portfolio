5Requirement Status Lifecycle

The status of each requirement evolves throughout the project lifecycle according to the following states:

StatusDescription
UNDEFINEDRequirement is tracked but not yet defined - awaiting stakeholder input or clarification
IDENTIFIEDInitial state - requirement has been elicited and documented
CONFIRMEDRequirement has been reviewed and agreed upon by all relevant parties
REVISIONEDRequirement has been superseded by a change request or merged into another requirement
CANCELEDRequirement has been removed - typically due to a change in scope or specifications
IMPLEMENTEDRequirement has been implemented in the system - may include a completion percentage
TESTEDRequirement has been verified against the implemented system
APPROVEDRequirement has been formally approved - release version indicated where applicable

7Non-Functional Requirements
7.1 Usability [UEX]

Requirements governing the quality of the user experience, interface consistency, accessibility, and learnability of the system.

[PRG]_UEX.001
[Usability requirement title] Rev. 1 IDENTIFIED
PriorityMUST
StabilityStable
SourceStakeholder, standard, or document that originated this requirement.
DescriptionDescribe the usability requirement - e.g. accessibility standard, response time perception, UI consistency rule.
Fit CriterionMeasurable or verifiable condition that confirms the requirement is satisfied - e.g. "The system must comply with WCAG 2.1 AA."
CommentsOpen notes or references.
7.2 Performance [PRF]

Requirements defining measurable performance thresholds such as response times, throughput, and capacity under load.

[PRG]_PRF.001
[Performance requirement title] Rev. 1 IDENTIFIED
PriorityMUST
StabilityStable
SourceStakeholder, standard, or document that originated this requirement.
DescriptionDescribe the performance requirement - e.g. maximum response time, concurrent user capacity, throughput threshold.
Fit CriterionMeasurable threshold - e.g. "p95 response time < 2s under a load of 500 concurrent users."
CommentsOpen notes or references.
