Software Requirement Specification — Enterprise & Consulting

An empty, reusable structure for enterprise and client-facing projects

This is a blank reusable template. It contains no project content. Its structure is independent of, and differs substantially from, the formal requirements adaptation published elsewhere on this site — this template uses its own requirement coding scheme, a revision and approval block, and a status lifecycle. Fields shown in [brackets] are placeholders; italic text describes what belongs in each field.

StatusDescription

The template defines eight lifecycle states in total, through to closure.
